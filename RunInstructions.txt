To run simulations:
condor_submit [name of file]

To check on simulations:
condor_q

To delete current simulations:
condor_rm [user]

To rebuild:
./arm_build.sh

Routing Algorithms:
0. TABLE
1. XY (for 2D Mesh topologies)
2. XYZ (for 2D/3D Mesh topologies)
3. XYZ_CHIPLETS (for user defined num_chiplet_x, num_chiplet_y configuration)
4. XYZ_CHIPLETS_HETERO (for standard 2x2 chiplet configuration)

Git:
1. git add .
2. git commit -m "commit message" -n
3. git push

To run more recent GCC version:
scl enable devtoolset-7 bash
